# ELT-with-Snowflake

##  📌 Project Overview 

### Business Problem:
Analyzes real-timeinsurance customers and claims data to identify 
high-risk regions, claim frequency trends, and cost exposure by policy type.

ThE task involved moving data from a NoSQL database (MongoDB) into cloud data warehouse (Snowflakes), using a transfer (Airbyte) 
transform the data using DBT and finally deriving insight via PowerBi dashboard.

Tech Stack:
- MongoDB (source / raw data store)
- Airbyte (ingestion)
- Snowflake (cloud data warehouse)
- dbt (transformation & data modeling)
- Power BI (reporting layer)

Skills Demonstrated:
- ELT pipeline design
- Dimensional data modeling (star schema)
- dbt testing & documentation
- Snowflake performance optimization
