# ELT-with-Snowflake

##  Project Overview 📌

### Business Problem:
Analyzes insurance claims data assets to identify 
high-risk regions, claim frequency trends, and cost exposure by policy type.

Tech Stack:
- MongoDB (source / raw data store)
- Airbyte (ingestion)
- Snowflake (cloud data warehouse)
- dbt (transformation & data modeling)
- Power BI (reporting layer)

Skills Demonstrated:
- ELT pipeline design
- Dimensional data modeling (star schema)
- dbt testing & documentation
- Snowflake performance optimization
